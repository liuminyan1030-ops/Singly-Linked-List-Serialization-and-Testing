# Singly Linked List: Serialization & Testing

## Project Overview
Implementation of a custom Singly Linked List data structure from scratch, focusing on memory efficiency, data persistence, and rigorous unit testing.

## Key Features
* **Custom Data Structure**: Built core operations including append, insert, and delete.
* **Object Serialization**: Implemented logic to save and load list structures to binary files.
* **Unit Testing**: Comprehensive test cases to verify functional correctness and edge cases.

## Tech Stack
* **Language**: Java
* **Topics**: Data Structures, Serialization, Unit Testing (JUnit)
